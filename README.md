# ultrasound_phantom
This project is about creating a cheap ultrasound phantom, to allow to test axial and lateral resolution of ultrasound devices.

The box is printed in PLA usig FDM printer
The lateral inner sides are printed in resin usine SLA printer

The wire to observe with ultrasound  has a spacing of 0.35 / 0.50 / 1.0 / 2.0 / 3.0

The holes in the lateral inner sides are made to pass inside a nylon wire of 0.1 diameter
The wire has to be pulled to be positionned in the closest angle to the next hole. 
We used some cyanolacrilate glue to keep the wires in position.

An absorber is put at the bottom of the box to avoid specular reflections. It's based on multiplying the reflections between triangle shapes. 

The gel used is some Agar (5%) and water. It has to be poured in the phantom at arround 50°c to avoid temperature deformation on the box. 

The main issue is to avoid bubbles. It's recomended to use a vacuum chamber.
